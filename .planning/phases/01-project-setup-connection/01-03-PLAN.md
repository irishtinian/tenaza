    ```kotlin
            val json = Json.parseToJsonElement(text).jsonObject

                "response" -> Json.decodeFromJsonElement<GatewayFrame.Response>(json)
                "event" -> Json.decodeFromJsonElement<GatewayFrame.Event>(json)

            }
    ```

    **ReconnectPolicy.kt** (package `com.clawpilot.data.remote.ws`):

    ```kotlin
    object ReconnectPolicy {
        private const val BASE_DELAY_MS = 1_000L
        private const val MAX_DELAY_MS = 30_000L
        private const val JITTER_MS = 500L

        fun delayFor(attempt: Int): Long {
            val exponential = BASE_DELAY_MS * (1L shl minOf(attempt, 4))
            val jitter = (Math.random() * JITTER_MS).toLong()
            return minOf(exponential, MAX_DELAY_MS) + jitter
        }
    }
    ```

    **WebSocketManager.kt** (package `com.clawpilot.data.remote.ws`):

    Class `WebSocketManager(private val okHttpClient: OkHttpClient, private val scope: CoroutineScope)`:

    **Public API:**
    - `val connectionState: StateFlow<ConnectionState>` — backed by `MutableStateFlow(Disconnected)`
    - `val frames: SharedFlow<GatewayFrame>` — backed by `MutableSharedFlow(replay = 0, extraBufferCapacity = 64)`
    - `fun connect(url: String, token: String)` — initiates WebSocket connection
    - `fun disconnect()` — user-initiated disconnect (no reconnect)
    - `fun send(frame: RequestFrame): Boolean` — serialize to JSON, send via webSocket.send()

    **Internal state:**
    - `private var webSocket: WebSocket? = null`
    - `private var reconnectJob: Job? = null`
    - `private var currentUrl: String? = null`
    - `private var currentToken: String? = null`
    - `private var reconnectAttempt: Int = 0`
    - `private var userDisconnected: Boolean = false`

    **connect(url, token) implementation:**
    1. Cancel any existing reconnectJob
    2. Store url and token
    3. Set `userDisconnected = false`
    4. Set state to `Connecting`
    5. **TLS enforcement (SECR-01):** If URL host is NOT `127.0.0.1`, `localhost`, or `10.x.x.x`/`192.168.x.x`, enforce `wss://`. Check: if url starts with `ws://` and host is not loopback/private, replace with `wss://`. Log a warning if URL was upgraded.
    6. Build OkHttp `Request` with URL: `$url/?token=$token`
    7. Create WebSocket: `okHttpClient.newWebSocket(request, webSocketListener)`
    8. Store reference to webSocket

    **WebSocketListener implementation (private inner class or lambda):**
    - `onOpen`: set state to `Connected`, reset `reconnectAttempt = 0`
    - `onMessage(text)`: call `parseGatewayFrame(text)` and emit to `_frames.tryEmit(frame)` if non-null
    - `onClosing`: call `webSocket.close(1000, "bye")`
    - `onClosed`: if `userDisconnected`, set state to `Disconnected`; else call `scheduleReconnect()`
    - `onFailure`: set state to `Error(throwable.message)`; if not `userDisconnected`, call `scheduleReconnect()`

    **disconnect() implementation:**
    1. Set `userDisconnected = true`
    2. Cancel reconnectJob
    3. `webSocket?.close(1000, "User disconnect")`
    4. `webSocket = null`
    5. Set state to `Disconnected`

    **scheduleReconnect() implementation:**
    1. If `userDisconnected`, return
    2. Increment `reconnectAttempt`
    3. val delay = `ReconnectPolicy.delayFor(reconnectAttempt)`
    4. Set state to `Reconnecting(reconnectAttempt, delay)`
    5. `reconnectJob = scope.launch { delay(delay); connect(currentUrl!!, currentToken!!) }`

    **send(frame) implementation:**
    1. val json = `Json.encodeToString(frame)`
    2. Return `webSocket?.send(json) ?: false`
