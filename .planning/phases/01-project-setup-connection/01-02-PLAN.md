    **KeyStoreManager.kt** (package `com.clawpilot.data.local.crypto`):

    Create a class `KeyStoreManager` with methods:

    1. `fun generateEcdsaKeyPair(alias: String = ALIAS): KeyPair`
       - Use `KeyPairGenerator.getInstance(KeyProperties.KEY_ALGORITHM_EC, "AndroidKeyStore")`
       - Initialize with `KeyGenParameterSpec.Builder(alias, PURPOSE_SIGN or PURPOSE_VERIFY)`
       - Set `.setAlgorithmParameterSpec(ECGenParameterSpec("secp256r1"))`
       - Set `.setDigests(KeyProperties.DIGEST_SHA256, KeyProperties.DIGEST_SHA512)`
       - Set `.setUserAuthenticationRequired(false)` (no biometric gate for device key)
       - Return `kpg.generateKeyPair()`

    2. `fun getPublicKeyBase64(): String`
       - Load KeyStore "AndroidKeyStore", call `load(null)`
       - Get certificate for ALIAS, extract `publicKey`
       - If no key exists, call `generateEcdsaKeyPair()` first
       - Return `Base64.encodeToString(publicKey.encoded, Base64.NO_WRAP)`

    3. `fun hasKeyPair(): Boolean`
       - Load KeyStore, check `containsAlias(ALIAS)`

    4. `fun deleteKeyPair()`
       - Load KeyStore, call `deleteEntry(ALIAS)`

    5. `fun sign(data: ByteArray): ByteArray`
       - Load private key from KeyStore
       - Use `Signature.getInstance("SHA256withECDSA")`
       - `initSign(privateKey)`, `update(data)`, return `sign()`

    Companion object: `private const val ALIAS = "clawpilot_device_key"`

    **CredentialStore.kt** (package `com.clawpilot.data.local.prefs`):

    Class `CredentialStore(private val context: Context)`:

    1. **Tink AEAD setup:**
       - Initialize Tink: `AeadConfig.register()` in an `init` block
       - Create `AndroidKeysetManager.Builder()`
         - `.withSharedPref(context, "clawpilot_tink_keyset", "clawpilot_tink_prefs")`
         - `.withKeyTemplate(KeyTemplates.get("AES256_GCM"))`
         - `.withMasterKeyUri("android-keystore://clawpilot_tink_master")`
         - `.build()`
       - Get `Aead` via `.keysetHandle.getPrimitive(Aead::class.java)`

    2. **DataStore instance:**
       - `private val Context.credentialDataStore by preferencesDataStore(name = "credentials")`

    3. **Preference keys (companion object):**
       - `val KEY_TOKEN = stringPreferencesKey("gateway_token")`
       - `val KEY_URL = stringPreferencesKey("gateway_url")`
       - `val KEY_DEVICE_NAME = stringPreferencesKey("device_name")`
       - `val KEY_SCOPES = stringPreferencesKey("scopes")`
       - `val KEY_PAIRED_AT = longPreferencesKey("paired_at")`

    4. **Store credentials:**
       ```kotlin
       suspend fun storeCredentials(credentials: GatewayCredentials) {
           context.credentialDataStore.edit { prefs ->
               prefs[KEY_TOKEN] = encrypt(credentials.token)
               prefs[KEY_URL] = encrypt(credentials.gatewayUrl)
               prefs[KEY_DEVICE_NAME] = credentials.deviceName
               prefs[KEY_SCOPES] = credentials.scopes.joinToString(",")
               prefs[KEY_PAIRED_AT] = credentials.pairedAt
           }
       }
       ```

    5. **Get credentials:**
       ```kotlin
       fun getCredentials(): Flow<GatewayCredentials?> =
           context.credentialDataStore.data.map { prefs ->
               val token = prefs[KEY_TOKEN]?.let { decrypt(it) } ?: return@map null
               val url = prefs[KEY_URL]?.let { decrypt(it) } ?: return@map null
               GatewayCredentials(
                   gatewayUrl = url,
                   token = token,
                   deviceName = prefs[KEY_DEVICE_NAME] ?: "",
                   scopes = prefs[KEY_SCOPES]?.split(",")?.filter { it.isNotEmpty() } ?: emptyList(),
                   pairedAt = prefs[KEY_PAIRED_AT] ?: 0L
               )
           }
       ```

    6. **Clear credentials (for unpair):**
       ```kotlin
       suspend fun clearCredentials() {
           context.credentialDataStore.edit { it.clear() }
       }
       ```

    7. **isPaired:**
       ```kotlin
       fun isPaired(): Flow<Boolean> =
           context.credentialDataStore.data.map { it[KEY_TOKEN] != null }
       ```

    8. **Private encrypt/decrypt helpers:**
       - `encrypt(value: String): String` — `aead.encrypt(value.toByteArray(Charsets.UTF_8), null)` -> Base64.NO_WRAP encode
       - `decrypt(encrypted: String): String` — Base64 decode -> `aead.decrypt(bytes, null)` -> String(UTF_8)
