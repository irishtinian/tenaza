    **PairingPayload.kt** (package `com.clawpilot.domain.model`):
    ```kotlin
    @Serializable
    data class PairingPayload(
        val v: Int,      // protocol version (expect 1)
        val u: String,   // gateway URL (e.g., "https://gw.example.com:18789")
        val t: String,   // pairing token (short, single-use)
        val e: Long      // expiry as Unix timestamp (seconds)
    ) {
        fun isExpired(): Boolean = System.currentTimeMillis() / 1000 > e

        fun toWebSocketUrl(): String {
            // Convert https:// to wss://, http:// to ws://
            return u.replace("https://", "wss://").replace("http://", "ws://")
        }
    }
    ```
