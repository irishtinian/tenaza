  <action>
    **AppRoutes.kt:**
    Define sealed interface hierarchy using `@Serializable` annotation (required by Navigation 3):
    ```kotlin
    @Serializable
    sealed interface AppRoute {
        @Serializable data object Chat : AppRoute
        @Serializable data object Dashboard : AppRoute
        @Serializable data object Crons : AppRoute
        @Serializable data object Settings : AppRoute
        @Serializable data object Pairing : AppRoute
        @Serializable data object ManualUrl : AppRoute
    }
    ```

    **Theme.kt:**
    - `ClawPilotTheme` composable wrapping `MaterialTheme` with dynamic color support (`dynamicDarkColorScheme`/`dynamicLightColorScheme` on API 31+, fallback to custom palette)
    - Follow system dark/light mode

    **Color.kt:**
    - Primary: deep purple (#6750A4 or similar) matching agent/AI theme
    - Define both light and dark color schemes using `lightColorScheme()` and `darkColorScheme()`

    **Typography.kt:**
    - Default Material 3 typography (no custom fonts in v1)

    **AppModule.kt (Koin):**
    - Create `val appModule = module { }` — empty for now, ViewModels added in later plans
    - Export as top-level val

    **ClawPilotApp.kt:**
    - Extend `Application`
    - Override `onCreate()`: call `startKoin { androidContext(this@ClawPilotApp); modules(appModule) }`
    - Import from `org.koin.android.ext.koin.androidContext` and `org.koin.core.context.startKoin`

    **MainShell.kt:**
    - `@Composable fun MainShell()` — the main app scaffold
    - `Scaffold` with `bottomBar` containing `NavigationBar` with 4 `NavigationBarItem` entries
    - Icons: `Icons.Default.Chat` (Chat), `Icons.Default.Dashboard` (Dashboard), `Icons.Default.Schedule` (Crons), `Icons.Default.Settings` (Settings)
    - Track selected tab with `var selectedTab by remember { mutableStateOf<AppRoute>(AppRoute.Dashboard) }`
    - Content area: use `NavDisplay` from Navigation 3 with `entryProvider` block
    - Each tab entry renders a placeholder composable: a `Box(Modifier.fillMaxSize(), contentAlignment = Center)` with `Text("Tab Name")`
    - Use `rememberNavBackStack` per tab for independent backstacks (per RESEARCH.md pattern)

    **AppNavHost.kt:**
    - `@Composable fun AppNavHost()` — top-level navigation handler
    - Checks if user is paired (placeholder boolean for now, default `true` to show MainShell)
    - If paired: show `MainShell()`
    - If not paired: show Pairing screen (placeholder `Text("Pairing")` for now)

    **MainActivity.kt:**
    - Extend `ComponentActivity`
    - `onCreate`: call `enableEdgeToEdge()` + `setContent { ClawPilotTheme { AppNavHost() } }`
    - Import `androidx.activity.enableEdgeToEdge`
